# Event Check-in and Attendance System

This is a console program that uses prompts to guid a user through several options step by step, allowing them to;

- create events,
- register attendees,
- check attendees in,
- generate attendance reports,
- and view events and attendees.

## How to:

<i><b>To run the system, you must have Node.js v24.13.0 installed;</b>

however, aside from Node.js itself, this program has no dependencies.</i>

### Run the application

Open the root folder in your prefered command line, and enter `node main`.

### Run the tests

Open the root folder in your prefered command line, and enter `node --test`.

<i>npm test <b>does not</b> run the tests because there is no package.json due to having no dependancies outside of Node.js.</i>
